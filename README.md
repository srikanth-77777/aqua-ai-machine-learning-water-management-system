# Aqua AI Water Management Dashboard

A lightweight static dashboard showcasing an AI-driven water management and monitoring experience for the AquaTwin project.

## Overview

This project includes a simulated water utility dashboard built from HTML, CSS, and JavaScript. It demonstrates:

- Interactive network map with node status, hover tooltips, and failure simulation
- Real-time analytics charts for pressure, flow, water quality, and pump energy
- AI decision engine with recommendation logs and action cards
- Leak detection heatmap with simulated scanning and zone selection
- Scenario simulator with adjustable demand, climate stress, and infrastructure age
- Animated background and custom cursor effects for a polished presentation

## Files

- `aquatwin-landing.html` — Landing page for the demo (static HTML page)
- `aquatwin-demo.html` — Built demo page generated by `build.py`
- `build.py` — Build script that merges `html.txt`, `css.txt`, and `js.txt` into `aquatwin-demo.html`
- `html.txt` — HTML source template used by the build script
- `css.txt` — CSS source stylesheet used by the build script
- `js.txt` — JavaScript source for UI interactions, animations, and simulated analytics

## Getting Started

### Prerequisites

- Python 3.x (for the build script)
- A modern browser to open the generated HTML file

### Build and Run

1. Open a terminal in the project folder.
2. Run:

```bash
python build.py
```

3. Open `aquatwin-demo.html` in your browser.

## How it Works

- `build.py` reads `html.txt`, `css.txt`, and `js.txt`.
- It injects CSS and JavaScript into the HTML template.
- The output is a standalone `aquatwin-demo.html` file.

## Notes

- This repo is a front-end demo and does not include a back-end or real sensor integration.
- All analytics, AI responses, and leak detection elements are simulated in JavaScript.
